Program terenAI generuje plik tekstowy w formacie, który można załadować do Symulatora Maszyna EU07 https://store.steampowered.com/app/1033030/MaSzyna/.
Aby program mógł wykonać swoje zadanie, będzie potrzebował pliku eksportu z programu Rainsted https://rainsted.com/pl/Strona_g%C5%82%C3%B3wna, który zawiera dane
node track, którym nadano realistyczne profile pionowe oraz przesunięcia względem środka scenerii. Oprócz tego potrzebne będą dane NMT1 które można pobrać
z portalu Geoportal https://www.geoportal.gov.pl/, które będą wykorzystywane do generowania terenu w pobliżu torów, oraz dane NMT100, które będą wykorzystywane
w dalszej odległości od torów. Dane NMT100 są aktualnie dostępne na stronie https://dane.gov.pl/pl/dataset/792,numeryczny-model-terenu-o-interwale-siatki-co-najmniej-100-m.
Najważniejsze parametry programu (nazwy plików, odległości od torów), można zmieniać w pliku .ini.
